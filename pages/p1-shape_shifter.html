Source code on GitHub: Shape Shifter

Objective

The goal of this project was to develop a 2D action-platformer featuring a unique mechanic: the ability for the player character to shapeshift. Each form grants temporary, ability-specific powers that can be used to overcome level-based obstacles and puzzles. The mechanic is designed to encourage experimentation, timing, and adaptability.

Project Status

Prototype completed. The team successfully developed a playable prototype that demonstrates core gameplay mechanics, including character transformation, basic level design, and visual/audio feedback loops.

Project Timeline

  • Start date: 2024-11-16
  • Completion date: 2025-03-19

Technologies Used

  • Game Engine: Unity (C#)
  • Graphics & Tools: Aseprite, GIMP, custom-built animation pipeline
  • Version Control: Git (GitHub)

Team Structure

The development team consisted of four members with clearly defined roles:

  • Project Lead / Game Design: Overall coordination, mechanic design, visual direction
  • Admin / Level Design: Environment design, puzzle layout, testing framework
  • Programmer 1 / UI Design: Core gameplay systems, user interface implementation
  • Programmer 2 / Sound Design: Audio assets, transformation logic, scene transitions

Retrospective Notes

Initially, the team aimed to complete a full-featured vertical slice of the game. However, given the limited timeframe and the lack of prior game development experience within the team, the scope was adjusted mid-project. The revised goal focused on building a solid and testable prototype to validate the core gameplay mechanic and project feasibility.

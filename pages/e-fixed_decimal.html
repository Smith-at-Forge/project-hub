Fixed Decimal

This code was developed as a learning exercise to practice C++. The code you see has been preserved as originally written to reflect the learning process and thought patterns at the time.

Source code on GitHub: fixed_decimal.cpp

Problem Statement

The task was to build a custom C++ class that stores decimal values as fixed-point integers to avoid rounding errors in financial calculations.

Compilation

Use the following command to compile the code:

g++ -Wall -o fixed_decimal fixed_decimal.cpp
